# Housing Instability in NYC

## Research Question
Do increases in evictions correspond with increases in the number of people staying in homeless shelters in New York City?

## Project Description
This project analyzes the relationship between eviction trends and homelessness in NYC. Using NYC Open Data datasets, we compare monthly eviction counts with the average number of individuals staying in homeless shelters to better understand housing instability in the city.

## Datasets
1. **NYC DHS Daily Report**
   - Daily shelter population statistics
   - https://data.cityofnewyork.us/Social-Services/DHS-Daily-Report/k46n-sa2m

2. **NYC Evictions Dataset**
   - Records of executed evictions in NYC
   - https://data.cityofnewyork.us/City-Government/Evictions/6z8x-wfk4

## Goal
Identify trends between evictions and homelessness and provide recommendations that could help improve housing stability in New York City.
